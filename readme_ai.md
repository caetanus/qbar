# QBar
QBar é uma bar estilo waybar, inspirada na waybar, porém feita em Qt e suporta x11 e wayland, a barra suporta transparencia e extensões shell. 
a barra é feita com QWidget e tem applets feitos em qml, a parte QWidget suporta theming via QML dinamico, alguns applets são padrão:
Clock, Calendar, Tray, Title, Workspaces, Caffeine, Sound, Battery, XInput.
ao clicar no relógio, deve descer um dropdown com calendário
